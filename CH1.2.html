HTML stands for Hypertext Markup Language

You use a text editor or a code editor to create HTML pages.

The Document Object Model (DOM) is a tree-like structure the browser creates after parsing HTML code when it receives an HTML page. With all of the components and attributes specified in the HTML code, the DOM symbolizes the web page's structure. Parsing CSS is the second stage in the rendering process.
